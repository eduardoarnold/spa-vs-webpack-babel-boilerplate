## Boilerplate for SPA using Visual Studio Solution

- Ready to IE8
- Ready to Publish (Using VS Publish or MSBuild)

### Development
```sh
$ npm install               # install npm dependencies
$ npm start                 # start webpack watchers
$ gulp lint:eslint          # run codelint
```
